Resources

Here are some useful websites for you:

  • http://www2u.biglobe.ne.jp/~b-jack/bn/pken.html - you can put numbers in this website, and it will create goroawase for you. This is a really good website. I recommend that you use it. このサイトに数字を入れると、あなたの語呂合わせを作ってくれるんです。これは本当に良いサイトです。私はあなたに使うように勧めています。
  • https://terakoya.ameba.jp/a000001596/ - this website teaches you how to use goroawase. このサイトでは、語呂合わせの使い方を紹介しています。
  • https://seoi.net/goro/ - this is another website which can create goroawase for you. こちらもあなたの語呂合わせを作成することができるサイトです。
